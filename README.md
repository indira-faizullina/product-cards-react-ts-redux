# React + TypeScript + Vite + Redux

Приложение для управления списком товаров.

## Функциональность

- Просмотр списка товаров
- Добавление в избранное
- Удаление товаров
- Создание новых товаров
- Фильтрация по избранным товарам
- Адаптивный интерфейс

## Технологии

- **React + TypeScript**
- **Redux Toolkit**
- **React Router**
- **Vite**
- **CSS Modules**
- **GitHub Pages**

## Установка и запуск

```bash
# Клонирование репозитория
git clone https://github.com/indira-faizullina/product-cards-react-ts-redux.git

# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка проекта
npm run build

# Деплой на GitHub Pages
npm run deploy
